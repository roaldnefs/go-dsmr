## Introduction
Go library for reading Dutch Smart Meter Requirements (DSMR). This is a pure Golang implementation based on the DSMR.

## Usage
### Installation
```console
go get github.com/roaldnefs/go-dsmr
```

### Importing
```go
import "github.com/roaldnefs/go-dsmr"
```
